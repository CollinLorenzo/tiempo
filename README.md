## Install

To install dependencies you will need [yarn](https://yarnpkg.com/en/), so go get it if you don't have it yet, then run:

```sh
yarn
```

## Running

To start the development server:

```sh
yarn start
```

To run all the test suite in watch mode:

```sh
yarn test
```
